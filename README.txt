Instrukcja uruchomienia na Koyeb:
1) Zaloguj się do https://www.koyeb.com
2) Stwórz nową aplikację (Create App) -> wybierz 'Manual Upload' lub połącz GitHub.
3) Jeśli wgrywasz ZIP ręcznie: wybierz opcję 'Upload' i wrzuć ten plik ZIP.
4) Ustaw Run Command: python bot.py
5) W sekcji Environment Variables dodaj:
   - DISCORD_TOKEN : (twój token bota Discord)
6) Deploy i poczekaj na start.
7) Panel webowy dostępny będzie pod portem 8080 (np. https://<your-app>.koyeb.app/)
   Hasło do panelu: piesfiga1
Pliki w pakiecie:
- bot.py         (główny skrypt bota)
- requirements.txt
- Procfile
